1. ภาพรวมของระบบ

ระบบใบสั่งซื้อ (Purchase Orders) เป็นส่วนหนึ่งของระบบ IchoicePMS ที่ช่วยจัดการกระบวนการสั่งซื้อสินค้าจากผู้จำหน่าย

ฟีเจอร์หลัก:

  • check_circleสร้างและจัดการใบสั่งซื้อ
  • check_circleติดตามสถานะการสั่งซื้อ
  • check_circleจัดการรายการสินค้าในใบสั่งซื้อ
  • check_circleคำนวณราคาและสกุลเงินอัตโนมัติ
  • check_circleพิมพ์และส่งออกข้อมูล
  • check_circleค้นหาและกรองข้อมูล

สถานะใบสั่งซื้อ:

รอดำเนินการ รับสินค้าบางส่วน เสร็จสิ้น ยกเลิก

2. การเริ่มต้นใช้งาน

1
เข้าสู่ระบบ

เข้าสู่ระบบ IchoicePMS ด้วยชื่อผู้ใช้และรหัสผ่านของคุณ

2
เข้าเมนูใบสั่งซื้อ

คลิกที่เมนู "ใบสั่งซื้อ" ในแถบเมนูด้านซ้าย

3
ตรวจสอบสิทธิ์การใช้งาน

ตรวจสอบว่าคุณมีสิทธิ์ในการสร้าง แก้ไข หรือลบใบสั่งซื้อ

lightbulbเคล็ดลับ: หากคุณใช้งานครั้งแรก ควรทำความคุ้นเคยกับส่วนต่างๆ ของหน้าจอก่อนเริ่มสร้างใบสั่งซื้อ

3. การดูรายการใบสั่งซื้อ

หน้าหลักจะแสดงตารางรายการใบสั่งซื้อทั้งหมด โดยจัดเรียงจากใหม่ไปเก่า

ข้อมูลที่แสดงในตาราง:

คอลัมน์ รายละเอียด ตัวอย่าง
PO ID รหัสใบสั่งซื้อที่ระบบสร้างให้อัตโนมัติ 1, 2, 3, ...
เลข PO เลขที่ใบสั่งซื้อที่กำหนดเอง PO2025001, ORDER-001
วันที่สั่งซื้อ วันที่ที่สร้างใบสั่งซื้อ 14/10/2025
ยอดรวม ยอดเงินรวมทั้งหมดของใบสั่งซื้อ 15,500.00 บาท
สถานะ สถานะปัจจุบันของใบสั่งซื้อ รอดำเนินการ, เสร็จสิ้น
ผู้จำหน่าย ชื่อบริษัทหรือผู้จำหน่าย บริษัท ABC จำกัด

การใช้งานฟีเจอร์ค้นหา:

ค้นหาอัตโนมัติ: พิมพ์คำที่ต้องการค้นหาในช่องค้นหา ระบบจะกรองข้อมูลทันที

การเรียงลำดับ: คลิกที่หัวคอลัมน์เพื่อเรียงลำดับข้อมูล

การแบ่งหน้า: เลือกจำนวนรายการต่อหน้าและใช้ปุ่มเปลี่ยนหน้า

4. การสร้างใบสั่งซื้อใหม่

1
คลิกปุ่มสร้างใบสั่งซื้อใหม่

2
กรอกข้อมูลพื้นฐาน
  • เลขที่ใบสั่งซื้อ (หรือใช้ที่ระบบสร้างให้)
  • วันที่สั่งซื้อ
  • เลือกผู้จำหน่าย
  • เลือกสกุลเงินและอัตราแลกเปลี่ยน
3
เพิ่มรายการสินค้า
  • ค้นหาและเลือกสินค้า
  • กำหนดจำนวนและราคา
  • ระบุหน่วยนับ
4
ตรวจสอบและบันทึก

ตรวจสอบความถูกต้องของข้อมูลและคลิกปุ่มบันทึก

warningข้อควรระวัง: ตรวจสอบข้อมูลให้ครบถ้วนก่อนบันทึก เพราะการแก้ไขภายหลังอาจมีข้อจำกัด

5. การดูรายละเอียดใบสั่งซื้อ

การเปิดดูรายละเอียด:

คลิกปุ่ม ในแถวของใบสั่งซื้อที่ต้องการ

ข้อมูลที่แสดงใน Popup รายละเอียด:

1. ข้อมูลหัวใบสั่งซื้อ
  • เลขที่ใบสั่งซื้อ และวันที่
  • สถานะปัจจุบัน
  • สกุลเงินและอัตราแลกเปลี่ยน
2. ข้อมูลผู้ทำรายการ
  • ชื่อผู้สร้างใบสั่งซื้อ
  • แผนกที่สังกัด
3. ข้อมูลผู้จำหน่าย
  • ชื่อบริษัทผู้จำหน่าย
  • ข้อมูลติดต่อ (โทรศัพท์, อีเมล)
  • ที่อยู่
4. รายการสินค้า
  • รายชื่อสินค้าพร้อมรูปภาพ
  • รหัสสินค้า (SKU)
  • จำนวนและหน่วยนับ
  • ราคาต่อหน่วยและยอดรวม
5. หมายเหตุ และข้อมูลเพิ่มเติม
  • หมายเหตุของใบสั่งซื้อ
  • วันที่สร้างและแก้ไขล่าสุด
  • ยอดรวมทั้งหมด
printการพิมพ์: คลิกปุ่ม "พิมพ์" เพื่อพิมพ์ใบสั่งซื้อในรูปแบบที่เหมาะสมกับกระดาษ A4

6. การแก้ไขใบสั่งซื้อ

การเข้าสู่โหมดแก้ไข:

คลิกปุ่ม ในแถวของใบสั่งซื้อที่ต้องการ

ข้อมูลที่สามารถแก้ไขได้:

1. ข้อมูลพื้นฐาน
  • เลขที่ใบสั่งซื้อ
  • วันที่สั่งซื้อ
  • หมายเหตุ
2. ผู้ทำรายการ

คลิกปุ่ม ในส่วนข้อมูลผู้ทำรายการ

เลือกผู้ใช้งานใหม่จาก dropdown และคลิก "บันทึก"

3. ผู้จำหน่าย

คลิกปุ่ม ในส่วนข้อมูลผู้จำหน่าย

เลือกผู้จำหน่ายใหม่จาก dropdown และคลิก "บันทึก"

infoหมายเหตุ: การแก้ไขข้อมูลจะส่งผลต่อยอดรวมและการคำนวณอื่นๆ ในระบบ

7. การจัดการรายการสินค้า

การแก้ไขรายการสินค้าแบบ Inline

1
เข้าโหมดแก้ไขรายการ

คลิกปุ่ม

2
แก้ไขข้อมูลสินค้า
  • ชื่อสินค้า: พิมพ์เพื่อค้นหาสินค้าใหม่ (มี autocomplete)
  • จำนวน: เปลี่ยนจำนวนสินค้า
  • ราคา: ปรับราคาต่อหน่วย
3
บันทึกหรือลบรายการ

4
เสร็จสิ้นการแก้ไข

คลิกปุ่ม

การเพิ่มสินค้าใหม่

คลิกปุ่ม

กรอกข้อมูลในฟอร์ม popup:

  • ชื่อสินค้า (มี autocomplete)
  • จำนวน
  • ราคาต่อหน่วย
  • หน่วยนับ
  • สกุลเงิน (หากแตกต่างจากใบสั่งซื้อ)
calculateการคำนวณอัตโนมัติ: ยอดรวมจะคำนวณและอัพเดทอัตโนมัติเมื่อมีการเปลี่ยนแปลงจำนวนหรือราคา

8. การจัดการสถานะใบสั่งซื้อ

สถานะต่างๆ และความหมาย:

สถานะ ความหมาย การดำเนินการที่เป็นไปได้
รอดำเนินการ ใบสั่งซื้อที่เพิ่งสร้างใหม่หรือยังไม่ได้รับสินค้า แก้ไข, ลบ, รับสินค้า
รับสินค้าบางส่วน ได้รับสินค้าแล้วบางรายการ แต่ยังไม่ครบ รับสินค้าเพิ่ม, ปรับสถานะ
เสร็จสิ้น ได้รับสินค้าครบถ้วนแล้ว ดูประวัติ, พิมพ์รายงาน
ยกเลิก ใบสั่งซื้อที่ถูกยกเลิก ดูประวัติเท่านั้น

การเปลี่ยนสถานะ:

สถานะจะเปลี่ยนแปลงอัตโนมัติตามการดำเนินงาน:

  • เมื่อรับสินค้าครั้งแรก → เปลี่ยนเป็น "รับสินค้าบางส่วน"
  • เมื่อรับสินค้าครบทุกรายการ → เปลี่ยนเป็น "เสร็จสิ้น"
  • เมื่อผู้ใช้ยกเลิกใบสั่งซื้อ → เปลี่ยนเป็น "ยกเลิก"

9. เคล็ดลับและข้อแนะนำ

การใช้งานที่มีประสิทธิภาพ:

searchการค้นหาสินค้า
  • พิมพ์เพียงบางส่วนของชื่อสินค้าหรือ SKU
  • ใช้คีย์บอร์ดลูกศรขึ้น-ลงเพื่อเลือกจากรายการ autocomplete
  • กด Enter เพื่อเลือกรายการที่ highlight
speedการทำงานที่รวดเร็ว
  • ใช้ Tab เพื่อเปลี่ยนระหว่างช่องกรอกข้อมูล
  • คลิกขวาที่ตารางเพื่อดูเมนูเพิ่มเติม (บางเบราว์เซอร์)
  • ใช้ Ctrl+F เพื่อค้นหาในหน้าปัจจุบัน
saveการบันทึกข้อมูล
  • บันทึกข้อมูลบ่อยๆ เพื่อป้องกันการสูญหาย
  • ตรวจสอบการเชื่อมต่ออินเทอร์เน็ตก่อนบันทึก
  • หากมีข้อผิดพลาด ให้รอสักครู่แล้วลองใหม่

ข้อควรระวัง:

  • ตรวจสอบข้อมูลให้ถูกต้องก่อนบันทึก
  • อย่าปิดหน้าต่างขณะที่กำลังบันทึกข้อมูล
  • สำรองข้อมูลสำคัญเป็นระยะ
  • ใช้งานบนเบราว์เซอร์ที่รองรับ (Chrome, Firefox, Edge)

10. การแก้ไขปัญหาเบื้องต้น

ปัญหาที่พบบ่อยและวิธีแก้ไข:

errorหน้าจอโหลดช้าหรือไม่ตอบสนอง

วิธีแก้ไข:

  • รีเฟรชหน้าเว็บ (F5 หรือ Ctrl+R)
  • ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
  • ลองใช้เบราว์เซอร์อื่น
  • ล้างแคชเบราว์เซอร์
errorไม่สามารถบันทึกข้อมูลได้

วิธีแก้ไข:

  • ตรวจสอบว่ากรอกข้อมูลครบถ้วน
  • ตรวจสอบรูปแบบข้อมูล (วันที่, ตัวเลข)
  • ลองบันทึกอีกครั้งหลังรอสักครู่
  • ออกจากระบบแล้วเข้าใหม่
errorข้อมูลไม่แสดงหรือแสดงผิด

วิธีแก้ไข:

  • รีเฟรชหน้าเว็บ
  • ตรวจสอบสิทธิ์การเข้าถึงข้อมูล
  • ล้างแคชและคุกกี้ของเบราว์เซอร์
errorPopup ไม่เปิดหรือมีปัญหา

วิธีแก้ไข:

  • ตรวจสอบว่าเบราว์เซอร์ไม่ได้บล็อค popup
  • ปิดโปรแกรมป้องกันไวรัสที่อาจบล็อค JavaScript
  • ใช้โหมด Incognito/Private browsing

การติดต่อขอความช่วยเหลือ:

หากยังไม่สามารถแก้ไขปัญหาได้ กรุณาติดต่อ:

  • ทีม IT Support ของบริษัท
  • ผู้ดูแลระบบ IchoicePMS
  • บันทึกหน้าจอปัญหาเพื่อประกอบการแก้ไข
